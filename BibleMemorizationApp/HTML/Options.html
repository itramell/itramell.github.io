If you like time trials, you'll enjoy this mode that includes a time limit to complete the game. If time runs out, the game will end!

Like singing the alphabet backwards? This mode has you stack words in reverse order, last word to first word, bottom to top. Think backwards songbirds!

Test your memory! This mode adds extra words that are not part of the memory verse to the word bank. Good luck!

This is for those who wants the most challenging of challenges! There is no reveal answer button, and if you get a word wrong you start over from the beginning. Good luck challenger!
